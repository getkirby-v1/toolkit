Function: go

Go to a given url

Syntax

go('url');

go('url');

Arguments

  1. url - (string) relative or absolute url

Example

go('http://google.com')
// sends the browser to the google homepage

go('/someotherpage')
// sends the browser to someotherpage (relative url)
